El catálogo

Nuestro portal usa la plataforma CKAN en su versión 2.5. CKAN es una herramienta desarrollada por la Open Knowledge Foundation. Se trata de software libre distribuido bajo la licencia GNU Affero General Public License.

Actualmente es una de las plataformas más populares para este tipo de iniciativas. La usan, entre otros, el Gobierno de Reino Unido, Estados Unidos, Brasil y Chile.

CKAN permite la organización de los datos publicados a través de su esquema de datasets y recursos, así como el acceso programático a éstos, aplicando estándares aprobados internacionalmente para la generación de metadata.

Metadata

Para hacer más fácil la navegación en el portal y la búsqueda de conjuntos de datos, usamos los estándares de metadata propuestos por defecto en CKAN (dublin core y dcat).

Integrar nuestros datos

Podés integrar los datos y los metadatos de un dataset por medio de la API de CKAN.

Esta es la forma genérica de consultar los metadatos: 

http://datos.gob.ar/api/3/action/package_show?id=<id_del_dataset>

Por ejemplo, podríamos ver la metadata del dataset de Estructura orgánica y autoridades del Poder Ejecutivo Nacional, consultando: 

http://datos.gob.ar/api/3/action/package_show?id=estructura-organica-pen

A su vez, esta es el modo genérico de consultar los datos de un recurso dentro de un dataset específico: 

http://datos.gob.ar/api/3/action/datastore_search?resource_id=<id_del_recurso>

¿Un ejemplo? Podemos ver los datos de un recurso del dataset de Estructura orgánica y autoridades del Poder Ejecutivo Nacional, así: 

http://datos.gob.ar/api/3/action/datastore_search?resource_id=b705d8c1-650f-43cc-bde6-68850fcecd21

Mirá más detalles para usar la API.
